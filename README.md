# OOD-KSHAP

This repository accompanies the IJCB 2026 paper **“Explaining Free-Text Keystroke Verification with OOD-aware SHAP”**.

The repository will contain:

- The implementation of the proposed OOD-aware SHAP framework
- Scripts and configurations for reproducing the experiments
- Supplementary materials and additional results

The official code and supplementary materials are being prepared and will be uploaded as soon as possible.
